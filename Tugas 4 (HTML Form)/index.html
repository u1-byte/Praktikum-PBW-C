Buatlah sebuah halaman web untuk login dan register untuk mahasiswa. Dalam halaman register minimal harus berisi:

  1. Input type text (bisa untuk email, nama, alamat, dll)
  2. Input type password (bisa untuk password)
  3. Input textarea (bisa untuk keterangan)
  4. Input type radio (bisa untuk jenis kelamin)
  5. Input type checkbox
  6. Select (bisa untuk agama)
  7. Upload foto

Halaman login harus berisi:

  1. Email
  2. Password
